## Iniciar

- Observer o arquivo axios.ts na linha 4. Ele contem a URL da API backend
```js
axios.defaults.baseURL = 'http://localhost:8000/api'
```
### Instalar os pacotes

```sh
npm install
```

### Executa em ambiente DEV

```sh
npm run dev
```

### Compilar 

```sh
npm run build
```

### Lint

```sh
npm run lint
```
